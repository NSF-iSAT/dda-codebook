## Definition

**Used when**: evidence is provided to cause the hearer to believe a claim. The evidence tend to be described in an **objective** fashion


## Examples

```
Example 1:
(1)A: Go to the next task 
(2)A: because you got to the end of the first one
(2) --Motivating--> (1)
```

```
Example 2:
(1)A: if you don't follow the instructions
(2)A: you won't get it right
(2) --Motivating--> (1)
```

```
Example 3:
(1)A: The reason it's not working
(2)A: you're not telling it what to do
(2) --Motivating--> (1)
```
