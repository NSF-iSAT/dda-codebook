## Definition

Used when: the connective indicates that the situation in the source argument follows/occurs after the situation described in target argument.

## Examples

```
(2) A: we are going to start with an evaluation first
(3) A: and then we form small groups.
---
(3) --After--> (2)
```
Forming small groups occurs after the evaluation.
