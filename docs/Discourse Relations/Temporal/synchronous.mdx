## Definition

Used when: there is some degree of temporal overlap between the events described by the arguments. All forms of overlap are included.

## Examples

```
Example 1:
(1) A: Jessie left the package on the porch
(2) A: while checking out if Lora was home.
---
(2) --Synchronous--> (1)
```
Simultaneously, Jessie left the package on the porch while checking out if Lora was home.
