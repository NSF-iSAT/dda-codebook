## Definition

**Used when:** one or more similarities between Arg1 and Arg2 are highlighted with respect to what each argument predicates as a whole or to some entities it mentions.

i.e. *This* is similar to *That*

## Examples

```
(1) A: We could use it to communicate underwater.
(2) B: Like dolphins.
---
(2) -- Similarity --> (1)
```
B highlights the similarity between the communication method described by A and the way dolphins communicate underwater.
