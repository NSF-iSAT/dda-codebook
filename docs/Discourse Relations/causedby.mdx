## Definition

**Used when:** the source argument provides the effect, while the target gives it's cause, reason, explanation or justification.

i.e. *This* is caused by *That*

## Examples

```
Example 1:
(1) A: Ah I think you have too many
(2) B: Yea I need to discard
(2) --CausedBy--> (1)
```
