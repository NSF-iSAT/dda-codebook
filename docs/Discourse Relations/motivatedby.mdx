## Definition

**Used when:** evidence is provided to cause the hearer to believe a claim. The evidence tend to be described in an objective fashion. 
`MotivatedBy` is used when the the source argument provides a claim which is motivated by the evidence in the target argument.

i.e. *This* is motivating *That*

## Examples

```
Example 1:
(1) A: I'm 5 foot 2
(2) B: then you're not tall
(2) --MotivatedBy--> (1)
```
