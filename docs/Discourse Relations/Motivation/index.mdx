## Definition

**Used when:** evidence is provided to cause the hearer to believe a claim. The evidence tend to be described in an objective fashion.

### Subtypes
- [Motivating](./motivating.mdx) is used when the source argument provides the evidence for the claim in the target argument. (i.e. *This* is motivating *That*)
- [MotivatedBy](./motivatedby.mdx) is used when the the source argument provides a claim which is motivated by the evidence in the target argument. (i.e. *This* is motivated by *That*)
