## Definition

**Used when:** the source argument gives the manner in answering 'how' the situation described in the target argument is done.

i.e. *This* is giving the manner of *That*

## Examples

```
(1) A: Get rid of that.
(2) A: By clicking on 'delete'.
---
(2) -- GivingManner --> (1)
```
(2) describes the manner (clicking delete) in which (1) is done (getting rid of something).

```
(922) A: Okay, now, correct it, guys, correct it.
(923) A: Like, fix all the lines and stuff.
---
(923) -- GivingManner --> (922)
```
(923) describes 'how' (fixing lines) to achieve the situation described in (922) (correcting it).
