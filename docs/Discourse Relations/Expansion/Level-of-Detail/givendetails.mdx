## Definition

**Used when:** the source argument describes a situation for which more details are provided in the target argument.

i.e. *This* is given details by *That*

## Examples

```
(1) A: The red one,
(2) A: I like that shirt.
---
(2) -- GivenDetails --> (1)
```
(1) provides details about the shirt described in (2).
