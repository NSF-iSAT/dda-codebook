## Definition

**Used when:** the source argument evokes a set of **circumstances** in which the described situation holds, and the target argument indicates one or more **exceptions** where it doesn't.

i.e. *This* is excluding *That*

## Examples

```
(1) A: Dolphins can't breath underwater.
(2) B: A lot of other sea creatures can.
---
(2) --Excluding--> (1)
```
