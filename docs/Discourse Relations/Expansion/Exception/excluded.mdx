## Definition

**Used when:** the source argument describes one or more **exceptions** which are excluded from the set of **circumstances** described by the target argument.

i.e. *This* is excluded from *That*

## Examples

```
(1) A: Sea creatures can breath underwater.
(2) B: Not dolphins and whales.
—-
(2) --Excluded--> (1)
```
the natural reading of the DDA edge: “Not dolpins and whales” is excluded from “Sea creatures can breath underwater.”
