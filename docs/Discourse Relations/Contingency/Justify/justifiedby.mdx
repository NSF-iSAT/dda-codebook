## Definition

**Used when:** the source argument is a situation or utterance justified by a **subjective or speaker centric** reason provided in the target argument.

i.e. *This* is justified by *That*

## Examples

```
Example 1:
(1) A: Do you want to trade?
(2) B: I don't want what you have.
(3) B: I think I'll pass.
(3) -- JustifiedBy --> (2)
```
