## Definition

**Used when:** the situations described in Arg1 and Arg2 are causally influenced but are not in a conditional relation.

### Subtypes
- [Causing](././causing.mdx) is used when the source argument causes or is causing the situation in the target argument. (i.e. *This* is causing *That*)
- [CausedBy](./causedby.mdx) is used when the the source argument is caused by the situation in the target argument. (i.e. *This* is caused by *That*)

## Example:
```
A: the sensor did not light up
A. since the battery is dead.
---
(2) --Causing--> (1)
```
