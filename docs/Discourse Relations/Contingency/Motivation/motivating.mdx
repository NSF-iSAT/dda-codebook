## Definition

**Used when:** evidence is provided to cause the hearer to believe a claim. The evidence tend to be described in an objective fashion. 
`Motivating` is used when the source argument provides the evidence for the claim in the target argument.

i.e. *This* is motivating *That*

## Examples

```
Example 1:
(1) A: The reason it's not working
(2) A: you're not telling it what to do
---
(2) --Motivating--> (1)
```
