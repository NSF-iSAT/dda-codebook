## Definition

**Used when:** one argument provides the potential ability for the actions or situation in the second argument to occur.
It is not necessary that the actions or situation actually occur, only that they are now possible as a result of the first argument.

### Example

```
(1) A: so then the rain stopped,
(2) A: allowing us to finish the game.
---
(2) -- Enabled By --> (1)
```
The speaker has the potential ability to finish the game, *enabled by* the rain stopping.
