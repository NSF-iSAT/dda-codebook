## Definition

**Used when:** when the source argument is an expression of gratitude for some previous action or statement made by another participant.
Mostly "thank you" or "thanks".

## Examples


```
(1) A: Can you pass the pen?
(2) B: here you go
(3) A: Thanks
---
(3) -- Thanking --> (2)
```
