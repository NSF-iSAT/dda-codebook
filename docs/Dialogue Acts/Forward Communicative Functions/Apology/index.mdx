## Definition

**Used when:** the source argument is an expression of regret or remorse for a mistake, offense, or inconvenience caused to another person.
If the source argument is intended to seek permission for a future course of action, it can be considered an **Offer**.
For example, "Excuse me for a moment" in advance of temporarily leaving commits the speaker to leaving, conditioned on the addressee's agreement.
