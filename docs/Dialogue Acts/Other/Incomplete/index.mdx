## Definition

**Used when:** the source argument is a slash unit that does not form a fully understandable utterance or snippet of utterance.
This label should be used for slash units that are hard to decipher the intention of the speaker.

## Examples

```
(1) A: You should-
...
(2) B: So how was your day?
---
(1) -- Incomplete --> (1)
```
The slash unit (1) is an incomplete utterance that is never completed.
